Adding Books

  1. Obtain a CSV file with Barcodes in column 1 and ISBN number in column 2.
  2. Click "Choose File" and select the CSV file.
  3. Click "Add Books" to automatically add the books into the database.

Adding Users

  1. Obtain a CSV file with emails in column 1, passwords in column 2, names in colum 3, and status ("Student or "Teacher") in column 4
  2. Click "Choose file" and select the CSV file.
  3. Click "Add users" to add the users into the firebase.
