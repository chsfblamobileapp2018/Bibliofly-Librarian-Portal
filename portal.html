Bibliofly - Library Database Quick Setup Portal

How to use automatic book entry system

  • Obtain a .csv file with Barcodes(in column 1) and ISBN numbers (in column 2)
  • Click the button which says "Choose File" and add the .csv
  • Finally click "Add Books" which parses the csv and uploads the books to your personal library database.

How to use automatic account creation system

  • Obtain a .csv file of emails (in column 1), passwords (in column 2), names (in column 3), and finally status (either "Student" or "Teacher" account) in column 4.
  • Click the button which says "Choose File" and add the .csv
  • Finally click "Create Accounts" which parses the .csv, creates a new account, and adds the new users credentials to the Online Authentication Database.
